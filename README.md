# Cookies
Simple tool for getting Cookies of multiple Facebook IDs at Once, Coded By Farhan Ali

# Run in Termux
```bash
git clone https://github.com/farhaanaliii/Cookies
cd Cookies
python main.py
```
# Note
Install git and python first then run this tool
Install git
```
pkg install git
```
Install Python
```
pkg install python
```
